# 玄武 · Genbu

武林同萌傳（TTHOL）的非官方資料站 —— 快速查裝備、看流派排行、比屬性。

## 你可以做什麼

### 查裝備
輸入道具名稱或依類型 / 等級瀏覽。每件裝備會顯示：
- 屬性條（跟同類型裝備比較的相對強度）
- 所有流派下的分位（越右邊代表此流派下越強）
- 隨機附加屬性與觸發率
- 掉落怪物來源

### 看排行
各類型裝備（座騎、背飾⋯）依流派自動排名。可以：
- 切換內建流派（如：純力攻、內外兼修、根骨醫毒）
- 自己調整屬性權重，存成個人流派
- 一眼看出哪幾件是通用裝、哪幾件偏某流派專精

### 比裝備
把最多 5 件想挑的裝備丟進比較盤，格狀矩陣一次看完差異，最高值自動標記。
